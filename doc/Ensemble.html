Rassemble les fichiers correspondant à une même entité.

 

Bibliothèque permettant de récupéré une structure contenant l'ensemble des fichiers correspondant à une même entité.

 

int AddEnsemble(ListEns* list, char *name)

 

 

Fonction qui ajoute en ensemble de fichier à la liste existante

 

Paramètre(s) :
  1. list est un pointeur sur la liste des ensembles déjà connu.
  2. name est une chaine de caractère correspondant au mot clès de cette ensemble

Return
Un int représentant un code de sortie.

 

 

int RemoveEnsemble(ListEns* list)

 

 

Fonction qui retire l'ensemble en-tête de la liste

 

Paramètre(s) :
  1. list est un pointeur sur la liste dans laquelle on veut retitré l'élément en-tête.

Return
Un int représentant un code de sortie.
