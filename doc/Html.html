Ecrit un fichier html .

 

On a ici les fonctions permettant d'écrire un fichier html utilisant un fichier CSS construit avec les fonctions contenu dans Css.h.

 

FILE* newHtml(char* name)

 

 

Fonction créant un fichier html

 

Le fichier créé aura déja la balise en-tête "head" de complété et sa tête de lecture possitionnée après la balise ouvrante

 

Paramètre(s) :
  1. name est le nom du fichier html à créer .

Return
Un FILE* qui est un pointeur sur le fichier html.

 

 

void newBalise(File* file, int code, char* String)

 

 

Fonction écrivant une chaine de caractères dans une balise

 

Paramètre(s) :
  1. file est un FILE* correspondant au fichier html dans lequel on doit mettre la balise .
  2. code est un int correspondant à une balise .
  3. String est la chaine de caractère a mettre dans la balise

 

int closeHtml(FILE* file)

 

 

Fonction fermant un fichier créé par la fonction newHtml

 

Paramètre(s) :
  1. file est un pointeur sur un fichier créé par la fonction newHtml

Return
un int correspondant à un code de sortie

 

 

void putTab (FILE * file, int nb)

 

 

Fonction mettant un nombre de tabulations pour indenter le fichier html

 

Paramètre(s) :
  1. file est un pointeur sur un fichier créé par la fonction newHtml
  2. nb est le nombre de tabulations à insérer dans le fichier pour indenter proprement le code html

 

void putSpace (FILE * file, int nb)

 

 

Fonction mettant un nombre d'espaces contribuant au css

 

Paramètre(s) :
  1. file est un pointeur sur un fichier créé par la fonction newHtml
  2. nb est le nombre d'espaces à insérer dans le fichier pour clarifier la page html
