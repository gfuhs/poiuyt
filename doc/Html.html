Create an html file.

 

Functions with which the program can create an html file using a css file built with the functions wrote in Css.c

 

FILE* newHtml(char* name)

 

 

Function which create an html file

 

The created file will already have the "head" completed and his CUR will be after the opening tag

 

Paramètre(s) :
  1. name is the html filename.

Return
A FILE* which is a pointer on the created file.

 

 

void newBalise(File* file, int code, char* String)

 

 

Function which write a String in a tag

 

Paramètre(s) :
  1. file is a FILE * on the html file in which the program has to write the tag.
  2. code is an integer which correspond to a tag.
  3. String is the string to put in the tag

 

int closeHtml(FILE* file)

 

 

Close a file created by the newHtml function

 

Paramètre(s) :
  1. file is a pointer on a file created by the newHtml function

Return
an integer which corresponds to a special exit code

 

 

void putTab (FILE * file, int nb)

 

 

Put a special number of tabulations to indent the html file

 

Paramètre(s) :
  1. file is a pointer on an html file created by the newHtml function
  2. nb is the number of tabluations to insert

 

void putSpace (FILE * file, int nb)

 

 

Insert some spaces in the html file to improve the look of the html page

 

Paramètre(s) :
  1. file is a pointer on an html file created by the newHtml function
  2. nb is the number of spaces to insert
