Bibliothèque contenant les fonctions permettant de manipuler un ensemble de fichier.

 

FILE* fusionFile (char *fileC, char* fileH, FILE* fileHtml)

 

 

Fonction fusionnant les commentaires des fichiers c et h et qui fait la liste des fonctions dans le fichier html

 

Paramètre(s) :
  1. fileC est le nom du fichier c a fusionner
  2. fileH est le nom du fichier h a fusionner
  3. fileHtml est un pointeur sur le fichier HTML qui correspond a la documentation du fichier c et h

Return
Un FILE* qui est un pointeur sur le fichier temporaire résultant de la fusion des deux fichiers c et h.

 

 

int ordreFile(char* file, char* name)

 

 

Fonction qui détermine si le fichier contient la balise d'en-tête

 

Paramètre(s) :
  1. file nom du fichier à tester s'il contient la balise d'en-tête
  2. name est la variable où on stockera le nom de l'ensemble de fichier si on trouve la balise d'en-tête

Return
Retourne 1 si oui sinon 0

 

 

void copieCom(FILE* fin, FILE* fout, FILE* html)

 

 

Fonction copie les commentaires d'un fichier dans un autre et fait la liste des fonctions dans le fichier d'entrée

 

Paramètre(s) :
  1. fin pointeur sur un fichier d'entrée
  2. fout pointeur sur un fichier de sortie
  3. html pointeur sur le fichier HTML qui correspond a la documentation du fichier d'entré

 

FILE* fusionFile(char *fileC, char* fileH, FILE* fileHtml)

 

 

Fonction qui fusionne les commentaires de 2 fichiers un c et h

 

Paramètre(s) :
  1. fileC est le chemin vers le fichier c.
  2. fileH est le chemin vers le fichier h.
  3. fileHtml est un pointeur sur le fichier correspondant à la documentation du fichier c et h

Return
Renvoie un pointeur sur le fichier obtenue après fusions des commentaires du fichierc avec le fichier h

 

 

void close_fusionFile(FILE* file)

 

 

Fonction qui ferme un fichier fusionner et le détruit

 

Paramètre(s) :
  1. file est le pointeur sur le fichier fusionner
