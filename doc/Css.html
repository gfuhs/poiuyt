Ecrit un fichier CSS par défault.

On a ici les fonctions permettant d'écrire un fichier de style CSS prédéfinis pour une page html.

int makeCss(int theme, char* path)

Fonction crée une fichier CSS prédéfinis

Paramètre(s) :
  1. theme est le code d'un style prédéfinis .
  2. path est le chemin au répertoire où on veut placer le fichier css

Return Un int représentant un code de sortie.
