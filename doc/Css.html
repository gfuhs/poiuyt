Ecrit un fichier CSS par défaut.

 

On a ici les fonctions permettant d'écrire un fichier de style CSS prédéfini pour une page html.

 

void themeCreationD(FILE * fin)

 

 

Fonction créant un fichier.css qui contiendra le thème des pages html

 

Paramètre(s) :
  1. fin est le fichier dans lequel on doit écrire le code css qui sera appliqué aux pages html

 

void printReset(FILE * reset)

 

 

Fonction créant un fichier de reset pour le css

 

Fonction créant un fichier de reset pour le css afin d'optimiser le nombre de navigateurs prenant en compte notre theme css

 

Paramètre(s) :
  1. reset est le fichier dans lequel on doit écrire le code de reset css

 

int makeCss(int theme, char* path)

 

 

Fonction créant un fichier CSS prédéfini

 

Paramètre(s) :
  1. theme est le code d'un style prédéfini .
  2. path est le chemin au répertoire où on veut placer les fichiers css

Return
Un int représentant un code de sortie.
